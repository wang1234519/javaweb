**Java_web员工管理系统**
=======================


**功能**
--------

* 使用javaweb相关的知识实现员工信息的增删改查


**依赖**
--------

* `[mysql-connector-java-8.0.30.jar]`
* `jdbc.properties`
* `servlet-api.jar`
* `tomcat-api.jar`

**安装依赖**
-------------

* `apache-tomcat-11.0.15-windows-x64`
* `idea`
* `JDK-17`

**配置**
------

1. 在idea中打开项目，进入项目结构，在模块中添加依赖中的两个jar包
2. 在项目结构中的工件中点击web应用程序展开型，并点击源项目
3. 在运行界面中进行编辑配置，选择tomcat本地服务器，并修改对应的端口，然后在部署中添加工件，完成后修改应用程序上下文中的路径（自己想要的路径）即可运行

**使用**
------

1. 直接运行代码文件即可
2. 运行后会弹出浏览器界面，在浏览器界面中显示相应的功能

**常见的错误及其解决方法**
-----
1.  网页遇到404 NOT_FOUND
 * 检查浏览器路径是否正确，项目的根路径是否为项目的名称

2. 查询不到信息或者数据库报错
* 确认是否在自己运行的终端进行了数据库的建立，并更改相应代码中数据库的配置文件

3. 点击相应的功能时显示不出来
* 路径存在问题，修改路径
